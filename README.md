# User Management API

A simple ASP.NET Core Web API that supports:
- CRUD operations on users
- Input validation
- Middleware for logging, error handling, and authentication

Use token: `Bearer mysecrettoken` for authentication.

Endpoints:  
- `GET /api/user`  
- `GET /api/user/{id}`  
- `POST /api/user`  
- `PUT /api/user/{id}`  
- `DELETE /api/user/{id}`
